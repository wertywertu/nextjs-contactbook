# Contact Book

Web application for managing personal contacts.  

---

## Features

- User registration and authentication  
- Create, view, edit, and delete contacts  
- Contacts are displayed as cards containing:
  - Name  
  - Email  
  - Phone number  
- Each user can only see their own contacts  
- Session handling using cookies

---

## Built with

- **Frontend:** Next.js, TypeScript, Axios  
- **Backend:** JSON Server  
- **Hosting:** Render and Vercel
